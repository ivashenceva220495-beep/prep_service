# PrepHub — AI-powered Interview Preparation Platform

**PrepHub** — это комплексная платформа для подготовки к техническим собеседованиям, объединяющая обучающие материалы, AI-тренажер с реальной оценкой ответов и интерактивную карту знаний (релиз 2.0) для отслеживания прогресса.

###  Возможности сервиса: 
###  Управление контентом
| Модуль | Описание |
|--------|----------|
| **Статьи** | CRUD операции с поддержкой Markdown, тегов и встроенных изображений |
| **Code Snippets** | Хранение примеров кода с подсветкой синтаксиса (10+ языков) |
| **Диаграммы** | Визуализация архитектур через Mermaid.js с возможностью zoom |

###  AI Тренажер
- **Генерация вопросов** на основе выбранной позиции (Junior → Senior)
- **Интеллектуальная оценка** ответов по 3 критериям (правильность, полнота, структура)
- **Детальная обратная связь** с выделением сильных сторон и зон роста
- **История собеседований** с аналитикой прогресса

###  Карта знаний (в разработке)
- **Визуализация графа знаний** (Cytoscape.js, force-directed layout)
- **Отслеживание прогресса** по каждой теме (0-100%)
- **Приоритизация изучения** на основе зависимостей между темами

###  Безопасность
- JWT-аутентификация через Keycloak (OAuth2 / OIDC)
- Row-Level Security (изоляция данных между пользователями)
- HttpOnly cookies для защиты от XSS
